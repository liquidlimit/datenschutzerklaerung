1. Allgemeines

Diese App verarbeitet keine personenbezogenen Daten im Sinne der Datenschutz-Grundverordnung (DSGVO). Die Nutzung erfolgt vollständig ohne Registrierung, Anmeldung oder Speicherung von personenbezogenen Informationen wie Name, Adresse, Standort oder Nutzungsverhalten.

2. Technische Daten

Die App benötigt folgende Systemfunktionen, um ordnungsgemäß zu funktionieren:

  • Bluetooth: zur Kommunikation mit Geräten in der Nähe. Je nach Android-Version kann das System eine Standortfreigabe anfordern, die jedoch nicht von der App selbst verwendet wird.
  • Internetverbindung: zur Kommunikation mit einem externen Server über Socket.io. Dabei wird technisch bedingt die IP-Adresse verarbeitet. Diese Daten werden nicht gespeichert oder ausgewertet.

3. Keine Analyse, kein Tracking

Die App verwendet keine Analyse- oder Trackingdienste, keine Cookies und keine Werbung. Es werden keine personenbezogenen Daten an Dritte weitergegeben.
